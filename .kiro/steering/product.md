# Product Overview

TechTok is a TikTok-style vertical scrolling web application for engineers to efficiently browse and consume technical articles and blog posts. The app aggregates content from RSS feeds and user posts, presenting them in an engaging, mobile-first interface.

## Target Users
- Engineers
- Technical directors
- Students
- Career changers aspiring to become engineers

## Core Value Proposition
Enable busy engineers to catch up on technical information efficiently through bite-sized, summarized content in a familiar vertical scroll format.

## Key Features
- Vertical scroll feed (TikTok-style UI)
- Technical article summaries from RSS feeds and user posts
- Social interactions (likes, bookmarks, comments, shares)
- User follow functionality
- Hashtag-based categorization
- Technology stack filtering
- OAuth authentication (Google/GitHub)
- User profiles with activity history
- Link to original articles

## Product Goals
- Reduce time needed to stay current with tech trends
- Make technical content consumption more engaging
- Build a community around technical knowledge sharing
