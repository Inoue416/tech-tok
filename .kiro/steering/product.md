# Product Overview

TechTok is a TikTok-style vertical scrolling application for engineers to efficiently consume technical content. The app aggregates and summarizes technical articles from RSS feeds and user posts.

## Target Users
- Engineers
- Directors
- Students
- Career changers aspiring to become engineers

## Core Features
- Vertical scrolling feed (TikTok-style UI)
- Technical article summaries from RSS feeds
- User-generated posts (text and video)
- Social features: likes, bookmarks, comments, follows
- Hashtag categorization
- Technology stack filtering
- Authentication via Google and GitHub OAuth

## Content Types
- RSS entries from technical blogs
- User posts (text and video)
- Unified feed combining both sources

## Key User Flows
- Unauthenticated users see limited content (first 10 items) with login prompts
- Authenticated users access full feed with personalized content
- Users can interact with posts (like, bookmark, comment, share)
- Users can follow other users and filter by technology interests
